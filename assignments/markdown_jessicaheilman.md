## Advantages and disadvantages

| Advantages | Disadvantages |
| ---------- | ------------- |
| Easy to learn | Limited styling |
| Widely used | Another markup language to learn |

## Tips

* Keep a cheatsheet handy, like
  [this one](https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet)
  on GitHub.
* Practice.
* Use other people's Markdown files as references/ideas for things to do.
